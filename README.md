SomaFOOS (stands for Soma Free Opensource Operating System) - is my Operating System created by me, and this operating system is the part of my big project called SomaFOP (Soma Free Opensource Project).

SomaFOP contains three projects in it (for now) - SomaFOOP, SomaFODOS (Soma Free Opensource Disk Operating System) and the website.
Now I'm working on SomaFOOS, after I will start work on SomaFODOS, and after I will start work on website.

If you want to commit to my repository, there is the rule: DON'T COMMIT IF YOU ARE NOT A PROGRAMMER.

I want to create my operating system for old PC's and without internet (because it's almost impossible write internet support alone). It's 32-bit operating system and it has it's own file system - "FSfSP".
This operating system has no graphical installer, only utilities to install with.

Later I will add guide how to build my operating system and run it in virtual emulators and real machines.
